Orbisnap Interface

This topic describes the following components of the Orbisnap interface:


  • Menu Bar
  • Toolbar
  • Navigation Panel

Using the menus, toolbar, and navigation panel, you can perform many of the same operations on the virtual world that you can with the Simulink 3D Animation viewer. For further details on using this interface open your Web browser and go to:

http://www.mathworks.com/access/helpdesk/help/toolbox/sl3d/index.html

You can also download the PDF version of the Simulink 3D Animation User's Guide documentation from the MathWorks Web site:

http://www.mathworks.com/access/helpdesk/help/pdf_doc/sl3d/sl3d.pdf
