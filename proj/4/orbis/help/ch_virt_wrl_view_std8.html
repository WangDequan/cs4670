Viewing Prerecorded WRL Animations or Virtual Worlds

This topic assumes that you have a prerecorded WRL animation file or an existing virtual world file.


  1. Start Orbisnap. For example, in Windows double-click orbisnap.bat .
  1. Orbisnap is displayed.

  1. In Orbisnap, select File -> Open.
  1. A file browser is displayed.

  1. Browse to the directory that contains the prerecorded WRL animation file or virtual world you want to view.
  2. Select the virtual world or prerecorded WRL file you want to view.
  3. Click Open.
  1. The file is displayed. If the file is an animation file, the simulation begins.

  1. To close Orbisnap, select File -> Close.
