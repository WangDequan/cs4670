Installing Orbisnap

You can obtain Orbisnap in one of two ways:


  • Web site -- You can download Orbisnap from http://www.orbisnap.com/download.html.
  • Simulink 3D Animation product -- Orbisnap is distributed with the Simulink 3D Animation for MATLAB® and Simulink®. The collection of Orbisnap files includes the Orbisnap executable file and supporting files. These files are located under the Simulink 3D Animation orbisnap directory (for example, <MATLAB root>\toolbox\sl3d\orbisnap). No further installation is necessary, but you might want to copy the Orbisnap files to another location. See Copying Orbisnap to Another Location.

Adding Shortcuts or Symbolic Links

For convenience, you can create a shortcut (Windows) or symbolic link (UNIX) to the Orbisnap starter file.


  • In Windows Explorer, right-click orbisnap.bat and select Properties. You can start Orbisnap from either the shortcut or the original batch file.
  • In UNIX, use the ln -s command to create a symbolic link to orbisnap starting script.
