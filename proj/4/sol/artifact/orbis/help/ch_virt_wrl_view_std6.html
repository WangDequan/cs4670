Copying Orbisnap to Another Location

Orbisnap runs independently of MATLAB and the Simulink 3D Animation. This means that you can copy Orbisnap to another location or even another machine. The following is a general procedure on how to copy Orbisnap to another location.


  1. From a command line or GUI such as Windows Explorer, create a directory into which you want to copy Orbisnap.
  2. Copy all the files in the Orbisnap directory and its subdirectories. These files are likely located in:

    • The Simulink 3D Animation orbisnap directory, for example
      <MATLAB root>\toolbox\sl3d\orbisnap
    • The directory structure into which you downloaded the Orbisnap zip file from the Web
  3. Paste the files into the directory you created in step 1.
