What is Orbisnap?

Orbisnap is a free stand-alone multi-platform VRML97 viewer. You can use Orbisnap to:


  • View and navigate a VRML world. You can navigate, render, and otherwise visualize a VRML world.
  • View prerecorded WRL animation files. For example, you might want to show prerecorded animation files in a meeting at which you do not have access to MATLAB or the Simulink 3D Animation.
  • Remotely view, from a client machine, a virtual world loaded in a current session of the Simulink 3D Animation. For example, if you want to visualize a virtual world active in a Simulink 3D Animation session that is running on a computer in another part of the building, or across the network. This functionality allows you to remotely view a simulation, but not control it.
